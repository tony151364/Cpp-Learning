## 参数
- VisibleAnywhere
- EditInstanceOnly：只有实例化之后才能在场景中编辑
- VisibleInstanceOnly：只有实例化之后才能在场景中可见
- EditDefaultsOnly：蓝图编辑器中可以编辑
- 不要将指针的类型设置为EditAnywhere，否则在蓝图中会很难操作
